函数声明提升

理解函数声明和函数表达式的区别的重要一点就是理解函数声明提升

函数声明提升 是指在执行代码之前，会先读取函数声明

const _ = {
  once: once,
};

var once = function(func){
  var flag = false;
  var res;
  return function(){
    if(flag){
      return res;
    }
    else{
      flag = true;
      res = func();
      return res;
    }
  }
}

module.exports = _;

变量声明是提前了，但once: once的时候，once是undefined;
用函数表达式时的时候需要注意执行顺序
