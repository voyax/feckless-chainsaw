1. **假定（BX）=637DH，（SI）=2A9BH，位移量D=3237H，试确定在以下各种寻址方式下的有效地址是什么？**

**（****1****）立即寻址**

**（****2****）直接寻址**

**（****3****）使用****BX****的寄存器寻址**

**（****4****）使用****BX****的间接寻址**

**（****5****）使用****BX****的寄存器相对寻址**

**（****6****）基址变址寻址**

**（****7****）相对基址变址寻址**

**答：**（1）立即数寻址的有效地址是当前IP的内容；

（2）直接寻址，若使用位移量D=3237H进行，则有效地址为3237H；

（3）使用BX的寄存器寻址时，操作数在BX寄存器中，因此无有效地址；

（4）使用BX的间接寻址时，有效地址在BX寄存器中，即有效地址=637DH；

（5）使用BX的寄存器相对寻址的有效地址=（BX）+D=637DH+3237H=95B4H；

（6）基址变址寻址的有效地址=（BX）+（SI）=637DH+2A9BH=8E18H；

（7）相对基址变址寻址的有效地址=（BX）+（SI）+D=C050H；

**2.&nbsp;****写出把首地址为****BLOCK****的字数组的第****6****个字送到****DX****寄存器的指令。要求使用以下几种寻址方式：**

**（****1****）寄存器间接寻址**

**（****2****）寄存器相对寻址**

**（****3****）基址变址寻址**

**答：**（1）使用寄存器间接寻址，把首地址为BLOCK的字数组的第6个字送到DX寄存器的指令为：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV BX，BLOCK

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ADD BX，12

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV DX，[BX]

（2）使用寄存器相对寻址，把首地址为BLOCK的字数组的第6个字送到DX寄存器的指令为：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV BX，BLOCK

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV DX，[BX+12]

（3）使用基址变址寻址，把首地址为BLOCK的字数组的第6个字送到DX寄存器的指令为：

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV BX，BLOCK

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV SI，12

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; MOV DX，[BX+SI]

**3.&nbsp;****现有****(DS)=2000H****，****(BX)=0100H****，****(SI)=0002H****，****(20100H)****＝****12H****，****(20101H)=34H****，****(20102H)=56H****，****(20103H)=78H****，****(21200H)=2AH****，****(21201H)=4CH****，****(21202H)=B7H****，****(21203H)=65H****，试说明下列各条指令执行完后****AX****寄存器的内容。**

**（****1****）****MOV&nbsp; AX****，****1200H**

**（****2****）****MOV&nbsp; AX****，****BX**

**（****3****）****MOV&nbsp; AX****，****[1200H]**

**（****4****）****MOV&nbsp; AX****，****[BX]**

**（****5****）****MOV&nbsp; AX****，****[BX+1100]**

**（****6****）****MOV&nbsp; AX****，****[BX+SI]**

**（****7****）****MOV&nbsp; AX****，****[BX+SI+1100]**

**答：**（1）指令MOV&nbsp; AX，1200H执行完后AX寄存器的内容为1200H；

&nbsp; （2）指令MOV&nbsp; AX，BX执行完后AX寄存器的内容为0100H；

&nbsp; （3）指令MOV&nbsp; AX，[1200H]是将从物理地址=（DS）*10H+1200H=21200H开始的两个单元内容送AX，执行完后AX寄存器的内容为4C2AH；

&nbsp; （4）指令MOV&nbsp; AX，[BX]是将从物理地址=（DS）*10H+（BX）=20100H开始的两个单元内容送AX，故执行完后AX寄存器的内容为3412H；

&nbsp; （5）指令MOV&nbsp; AX，[BX+1100]是将从物理地址=（DS）*10H+（BX）+1100H=21200H开始的两个单元内容送AX，故执行完后AX寄存器的内容为4C2AH；

&nbsp; （6）指令MOV&nbsp; AX，[BX+SI]是将从物理地址=（DS）*10H+（BX）+（SI）=20102H开始的两个单元内容送AX，故执行完后AX寄存器的内容为7856H；

&nbsp; （7）指令MOV&nbsp; AX，[BX+SI+1100]是将从物理地址=（DS）*10H+（BX）+（SI）+1100H=21202H开始的两个单元内容送AX，故执行完后AX寄存器的内容为65B7H；

**4.****假设已知****(DS)=2900H,(ES)=2100H,(SS)=1500H,(SI)=00A0H,(BX)= 0100H****，****(BP)=0010H****，数据段中变量名****VAL****的偏移地址值为****0050H****，试指出下列源操作数字段的寻址方式是什么？其物理地址值是多少？**

&nbsp;&nbsp;&nbsp;**（****1****）**** MOV AX****，****0ABH&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****2****）**** MOV AX****，****BX**

&nbsp;&nbsp;&nbsp;**（****3****）**** MOV AX****，****[100H]&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****4****）**** MOV AX****，****VAL**

&nbsp;&nbsp;&nbsp;**（****5****）**** MOV AX****，****[BX]&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****6****）**** MOV AX****，****ES:[BX]**

&nbsp;&nbsp;&nbsp;**（****7****）**** MOV AX****，****[BP]&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****8****）**** MOV AX****，****[SI]**

&nbsp;&nbsp;&nbsp;**（****9****）**** MOV AX****，****[BX+10]&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****10****）**** MOV AX****，****VAL[BX]**

&nbsp;&nbsp;&nbsp;**（****11****）**** MOV AX****，****[BX][SI]&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ****（****12****）**** MOV AX****，****[BP][SI]**

**答：**（1）在指令 MOV AX，0ABH 中，源操作数字段的寻址方式是立即数寻址，其物理地址值=（CS）*10H+（IP）；

&nbsp; （2）在指令 MOV AX，BX 中，源操作数字段的寻址方式是寄存器寻址，操作数在BX中，无物理地址；

&nbsp; （3）在指令 MOV AX，[100H] 中，源操作数字段的寻址方式是直接寻址，其物理地址值=（DS）*10H+100 =29000H+100H=29100；

&nbsp; （4）在指令 MOV AX，VAL 中，源操作数字段的寻址方式是直接寻址，其物理地址值=（DS）*10H+50H =29000H+50H=29050H；

&nbsp; （5）在指令 MOV AX，[BX] 中，源操作数字段的寻址方式是寄存器间接寻址，其物理地址值=（DS）*10H +（BX）=29000H+100H=29100H；

&nbsp; （6）在指令 MOV AX，ES：[BX] 中，源操作数字段的寻址方式是寄存器间接寻址，其物理地址值=（ES）*10H+（BX）=21000H+100H=21100H；

&nbsp; （7）在指令 MOV AX，[BP] 中，源操作数字段的寻址方式是寄存器间接寻址，其物理地址值=（SS）*10H +（BP）=15000H+10H=15010H；

&nbsp; （8）在指令 MOV AX，[SI] 中，源操作数字段的寻址方式是寄存器间接寻址，其物理地址值=（DS）*10H +（SI）=29000H+0A0H=290A0H；

&nbsp; （9）在指令 MOV AX，[BX+10] 中，源操作数字段的寻址方式是寄存器相对寻址，其物理地址值=（DS）*10H+（BX）+0AH= 29000H+100H+0AH =2910AH；

&nbsp; （10）在指令 MOV AX，VAL[BX] 中，源操作数字段的寻址方式是寄存器相对寻址，其物理地址值=（DS）*10H+（BX）+50H= 29000H+100H+50H= 29150H；

&nbsp; （11）在指令 MOV AX，[BX][SI] 中，源操作数字段的寻址方式是基址变址寻址，其物理地址值=（DS）*10H+（BX）+（SI） =29000H+100H+0A0H =291A0H；

&nbsp; （12）在指令 MOV AX，[BP][SI] 中，源操作数字段的寻址方式是基址变址寻址，其物理地址值=（SS）*10H+（BP）+（SI）=15000H+10H+0A0H =150B0H

**5.****分别指出下列指令中的源操作数和目的操作数的寻址方式。**

&nbsp;&nbsp;&nbsp;**（****1****）****MOV&nbsp; SI****，****200**

&nbsp;&nbsp;&nbsp;**（****2****）****MOV&nbsp; CX****，****DATA[SI]**

&nbsp;&nbsp;&nbsp;**（****3****）****ADD&nbsp; AX****，****[BX+DI]**

**答：**（1）目的操作数字段的寻址方式是寄存器寻址，源操作数字段的寻址方式是立即数寻址；

（2）目的操作数的寻址方式是寄存器寻址，源操作数的寻址方式是寄存器相对寻址；

（3）目的操作数的寻址方式是寄存器寻址，源操作数的寻址方式是基址变址寻址；

（4）目的操作数的寻址方式是寄存器寻址，源操作数的寻址方式也是寄存器寻址；

（5）目的操作数的寻址方式是寄存器间接寻址，源操作数的寻址方式是寄存器寻址；

**6.****试述指令****MOV AX****，****2010H****和****MOV AX****，****DS:[2010H] ****的区别。**

**答：**指令MOV AX，2010H是将立即数2010H送AX寄存器，而指令MOV AX，DS:[2010H]是将DS段有效地址为2010H的两个单元的内容送AX。
